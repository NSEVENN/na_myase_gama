# Картинки предметов

Теперь проект не использует придуманные SVG-заглушки для карточек.

Как работает:
- если у предмета есть локальный `image` в `skins.js`, берется он;
- если локальной картинки нет, сайт автоматически загружает настоящие игровые иконки из ByMykel CSGO-API/CDN;
- поддерживаются скины, ножи, перчатки, наклейки, кейсы, капсулы, graffiti, patches, collectibles и music kits;
- результат кэшируется в браузере на 7 дней, чтобы сайт потом открывался быстрее;
- пока иконка грузится, показывается маленький CS2-loader, а не фейковая картинка.

Для GitHub Pages это самый удобный вариант: не нужно хранить тысячи картинок в архиве.

FIX: реальные картинки теперь грузятся не через внутренний JSON-кэш, а напрямую через redirect SteamApis:
https://api.steamapis.com/image/item/730/{market_hash_name}
Если ByMykel API не успел загрузиться/заблокирован, карточки всё равно получают настоящую Steam CDN картинку по названию предмета.
